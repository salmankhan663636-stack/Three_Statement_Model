## 📊 Project Overview
This project contains a dynamically linked, three-statement financial model for **Infosys**. It connects the historical performance data to future projections, ensuring that any changes in operational assumptions automatically flow through all three core financial statements seamlessly.

## 🔑 Key Features of the Model
* **Fully Integrated Statements:** The Income Statement, Balance Sheet, and Cash Flow Statement are completely tied together using dynamic Excel formulas (no hardcoded balances).
* **Automated Balancing:** The Balance Sheet utilizes automated checks to ensure total assets always equal total equity and liabilities across all forecast periods.
* **Supporting Schedules:** Includes core working schedules for Depreciation, Working Capital, and Debt/Interest tracking.

## 🛠️ Tools & Skills Demonstrated
* **Software:** Microsoft Excel
* **Financial Modeling:** Historical Trend Analysis, Financial Forecasting, Three-Statement Integration, and Financial Ratio Analysis
